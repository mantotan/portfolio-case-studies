### polymarket-copy-trade
Rust + TypeScript · 2026 — Polyglot trade-execution platform with Rust hot-path (zero DB reads), three-signal trade detection across independent failure domains, 21-step filter chain.
https://github.com/mantotan/polymarket-copy-trade

### crypto-watcher-bot-backend
NestJS + TypeScript + PostgreSQL · 2026 — Trading-bot API: auth refactored into 8 focused services, AES-256-GCM encryption, Redis Pub/Sub-backed WebSocket Gateway for backtest progress, hedging constraints enforced at the DB layer.
https://github.com/mantotan/crypto-watcher-bot-backend

### crypto-watcher-bot-frontend
Next.js 15 + React 19 + Tailwind v4 · 2026 — Dashboard for the trading-bot backend above. JWT auto-refresh interceptor, middleware-protected routes, real-time backtest progress via Socket.IO.
https://github.com/mantotan/crypto-watcher-bot-frontend

### quant-modelling
Python · 2026 — Quant ML pipeline trading crypto-binary prediction markets on Polymarket. LightGBM + Optuna HPO + treelite-compiled inference, walk-forward CPCV + PBO + isotonic calibration, fractional-Kelly risk. Multi-agent autonomous research loop (13 agent definitions, 150+ HPO iterations to structural-floor exhaustion).
