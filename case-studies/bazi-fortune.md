# Bazi Fortune - AI Astrology Product

## Summary

Bazi Fortune is an AI-powered mobile product that combines traditional Chinese Bazi / Four Pillars astrology with modern AI interpretation. The app helps users generate personalized birth charts, understand daily and hourly fortune, explore compatibility, select auspicious dates, and ask AI-guided questions about career, wealth, relationships, health, and timing.

Website: https://bazifortune.app

## Role

Built end-to-end, including:
- Product planning and feature design
- Backend architecture
- Mobile app development
- Landing page development
- AI feature design
- Hosting and deployment
- Iteration with AI-assisted development workflows

## Stack

- Mobile: React Native
- Landing page: Next.js
- Backend: Node.js, TypeScript, NestJS
- AI: AI-powered interpretation/chat flows
- Hosting: Self-managed deployment/hosting setup

## Key Features

- Personalized Bazi / Four Pillars birth chart generation
- Daily fortune insights for career, wealth, relationships, and health
- Hourly fortune based on traditional Chinese time periods
- Relationship compatibility analysis
- Auspicious date selection
- AI chat that understands the user's chart and context
- Multilingual support
- Mobile-first user experience

## Architecture Notes

The product combines deterministic domain logic for Bazi chart generation with AI-generated interpretation layers. The backend is responsible for user data flows, chart-related calculations, AI request orchestration, and product APIs. The mobile app focuses on structured presentation of complex metaphysical data in a simple consumer-facing experience.
